Protocols

Protocols

A collection of Internet protocols is a conceptual model and set of communication protocols used on the Internet and similar computer networks. It is commonly known as TCP / IP because the underlying protocols in the packet are Transmission Control Protocol (TCP) and Internet Protocol (IP).

HTTP/HTTPS

HTTP - hypertext transfer protocol It is the standard protocol for exchanging any kind of data on the Internet. HTTP use 80 port
HTTPS - hypertext transfer protocol secure. Used to protect confidential data and financial information transmitted over the Internet HTTPS use 443 port

HTTP/HTTPS response status codes

100 Information responses
200 Successful responses
300 Redirection messages
400 Client error responses
500 Server error responses

IP

An Internet Protocol address (IP address) is a numerical label assigned to each device connected to a computer network that uses the Internet Protocol for communication.[1][2] An IP address serves two main functions: host or network interface identification and location addressing.
